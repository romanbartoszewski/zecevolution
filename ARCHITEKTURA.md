# Architektura Zecevolucji

Projekt rozwijany jest jako dynamiczna ontologia.

Nie jest liniową narracją.
Jest siecią pojęć powiązanych relacjami.

Model:

- Węzły reprezentują pojęcia ontologiczne i mechanizmy.
- Relacje reprezentują zależności strukturalne i przejścia.
- Graf nie posiada pojedynczego centrum.
- Struktura jest sieciowa.

Docelowa implementacja obejmuje:

- interaktywną mapę relacji,
- dynamiczne rozszerzanie ontologii,
- wizualizację trajektorii reorganizacji,
- wersjonowanie zmian koncepcyjnych.

Zecevolucja jest traktowana jako system poznawczy,
a nie wyłącznie tekst teoretyczny.
