# docs/ — interfejs (GitHub Pages)

Ten katalog jest źródłem statycznego interfejsu projektu i jednocześnie katalogiem publikacji dla GitHub Pages.

- Strona: `index.html`
- Style: `style.css`
- Logika: `graph.js`
- Dane: `data.json` (oraz opcjonalnie `ontology.json`)

Zasada pracy:
- Edytuj interfejs wyłącznie w `docs/` (nie ma osobnego `interface/`).
- Każda zmiana w `docs/` jest wersjonowana przez historię commitów w repo.
